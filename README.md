# 🚀 PROJETOS UEG - Sistemas para Internet

★ Bem-vindo ao repositório central de projetos desenvolvidos durante as atividades acadêmicas do curso de **Sistemas para Internet – UEG Itaberaí**, sob orientação de professores da faculdade.

## 📝 Sobre o Repositório
Este espaço foi criado para centralizar, versionar e organizar o código-fonte de projetos de pesquisa, extensão, TCCs e protótipos experimentais. O objetivo é criar um portfólio de fácil acesso e uma base de conhecimento para estudantes.

## 📂 Índice de Projetos
Abaixo estão os projetos organizados por tecnologia e natureza:

| Nome do Projeto | Linguagem/Stack | Tipo | Status |
| :--- | :--- | :--- | :--- |
| **Portal de Projetos** | HTML, Tailwind, JS | Web / Institucional | ON |
| **Assistente de Edital UEG** | Python, Gemini API, ChromaDB | IA / RAG | ON |
| **Chuvisco Bot** | Python, Telegram API | Automação / Clima | OFF |

---

## 🛠 Tecnologias em Uso
Abaixo, algumas das tecnologias que você encontrará ao explorar este repositório:

*   **Front-end:** HTML5, CSS3 (Tailwind), JavaScript.
*   **Inteligência Artificial:** 
    *   **LLMs:** Google Gemini API (modelos Flash e Pro).
    *   **Arquitetura RAG:** Implementação de busca em documentos (PDF) com bases vetoriais.
    *   **Vector Databases:** ChromaDB para armazenamento de embeddings.
*   **Back-end & Bots:** Python, Telegram Bot API.
*   **DevOps & Infra:** Deploy via Render com monitoramento por Cron-jobs.
