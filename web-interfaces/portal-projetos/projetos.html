1. Projetos de Pesquisa

1.1 Modelagem Multissensores de Estoques de Carbono em Áreas de Restauração no Brasil com Sensoriamento Remoto e Inteligência Artificial

Projeto relacionado à pesquisa de doutorado do professor responsável, já cadastrado institucionalmente na UEG.

O objetivo é desenvolver modelos de estimativa de carbono acima do solo em áreas de restauração ecológica utilizando dados de sensoriamento remoto multissensor (satélites, radar, etc.) combinados com técnicas de inteligência artificial. Prevê possível integração com pesquisadores do LAPIG/UFG e com a TNC Brasil.

Vantagens

  • Tema de pesquisa científica de ponta
  • Forte potencial de publicações científicas
  • Colaboração com pesquisadores do LAPIG/UFG
  • Experiência com IA aplicada ao meio ambiente e sensoriamento remoto

Desafios

  • Alta complexidade técnica
  • Sem previsão de bolsa pela UEG no momento
  • Análise de dados ambientais complexos
Pesquisa IA Sensoriamento Remoto Dados Ambientais

1.2 Algoritmos em R para Monitoramento Ambiental: Automação de Fluxos de Dados e Desenvolvimento de Materiais para Transferência Tecnológica

Projeto em fase final de aprovação na UEG, com previsão de início das atividades em agosto de 2026.

O objetivo é desenvolver algoritmos e pipelines automatizados em linguagem R para processamento de dados ambientais, além da produção de materiais de transferência tecnológica. O projeto faz parte de uma iniciativa de maior escala envolvendo instituições nacionais.

Scripts reprodutíveis • Manuais técnicos • Documentação metodológica • Vídeos tutoriais

Vantagens

  • Projeto multidisciplinar de escala nacional (Eletrobrás)
  • Integração com equipe técnica qualificada
  • Experiência prática em automação de dados ambientais
  • Previsão de 1 bolsa de Iniciação Científica

Desafios

  • Complexidade técnica do projeto
  • Necessidade de aprendizado da linguagem R
  • Envolvimento com pipelines de dados complexos
Pesquisa Linguagem R Automação Dados Ambientais

2. Projetos de Extensão

2.1 Mapeando a Cidade

Projeto já realizado anteriormente, com produto final entregue e publicação associada. Nova edição prevista possivelmente em parceria com a equipe de turismo de Caldas Novas.

Construção colaborativa de mapas digitais de cidades turísticas, integrando informações culturais, históricas e geográficas utilizando plataformas de mapeamento digital.

Ver projeto oficial

Vantagens

  • Fácil entrada para estudantes
  • Uso de ferramentas acessíveis (Google MyMaps)
  • Experiência prática em cartografia digital
  • Participação em atividades de extensão

Desafios

  • Dependência de dados fornecidos pela equipe de turismo
  • Participação institucional externa necessária
Extensão Mapeamento Web Dados Geoespaciais

2.2 Chatbot com IA para a Plataforma "LAPIG na Escola"

Projeto em andamento no ano atual, sendo desenvolvido em conjunto com um estudante de TCC.

O objetivo é criar um chatbot com Inteligência Artificial integrado à plataforma educacional do LAPIG, facilitando o acesso a conteúdos educacionais sobre geografia, sensoriamento remoto e meio ambiente.

Vantagens

  • Possibilidade de cumprimento de horas de extensão
  • Experiência prática com agentes de IA
  • Integração com plataforma educacional real

Desafios

  • Sem previsão de bolsa no momento
  • Produto principal já sendo entregue pelo aluno de TCC envolvido
Extensão IA Chatbots Tecnologias Educacionais
