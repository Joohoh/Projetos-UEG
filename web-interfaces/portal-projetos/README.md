# Portal de Projetos Acadêmicos

Portal estático desenvolvido para divulgar oportunidades de pesquisa, extensão e projetos externos para estudantes do Curso de Sistemas para Internet da **UEG – Polo Itaberaí**.

---

## 📄 Páginas

| Arquivo | Descrição |
|---|---|
| `index.html` | Página inicial com visão geral dos projetos |
| `projetos.html` | Detalhamento das oportunidades ativas |
| `ideias.html` | Áreas de interesse e envio de propostas |

---

## 🛠️ Tecnologias e Ferramentas

### Interface
- **[Tailwind CSS](https://tailwindcss.com/)** — estilização via classes utilitárias, incluindo dark mode e responsividade
- **[Lucide Icons](https://lucide.dev/)** — biblioteca de ícones SVG

### Editor e Extensões
- **[VS Code](https://code.visualstudio.com/)** — editor de código
- **[GitHub Copilot](https://github.com/features/copilot)** — assistente de código com IA integrado ao VS Code

### Design
- **[Figma](https://figma.com/)** — prototipação do layout antes do desenvolvimento

### IA Generativa
- **[Claude (Anthropic)](https://claude.ai/)** — apoio na revisão de código, boas práticas e estrutura HTML
